# Cyber Attack Detection in Smart Digital Substations
Machine learning-based cyber attack detection framework for IEC 61850 smart digital substations.

This repository contains the IEEE conference paper presented at IDCIoT 2026.

-----

## Author
Aybuke Senel  
Computer Engineering – Halic University

-----

## Conference
4th International Conference on Intelligent Data Communication Technologies and Internet of Things (IDCIoT 2026)

-----

## Overview
This research proposes a machine learning framework for detecting simultaneous cyber attacks in IEC 61850 based smart digital substations.

The study focuses on analyzing GOOSE protocol communication data and applying advanced feature engineering techniques together with machine learning and deep learning models.

-----

## Methods
The following models were evaluated:

- Random Forest
- XGBoost
- LightGBM
- CatBoost
- Convolutional Neural Networks (CNN)
- Long Short-Term Memory Networks (LSTM)
- Hybrid CNN–LSTM architecture

-----

## Results
The proposed Hybrid CNN-LSTM model achieved 99.72% accuracy in detecting replay, masquerade, and denial-of-service attacks.
